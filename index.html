Osobní vzkazy

Jak automatizovat dashboard

3 kroky od manuálního uploadu k live datům

1
Tally → Google Sheets (nativní napojení)
V Tally přejdi na Integrations → Google Sheets a vyber nebo vytvoř sheet. Každé odeslání formuláře se automaticky zapíše jako nový řádek — žádný webhook ani Make není potřeba pro základní sběr.
Tally → Integrations → Google Sheets
2
Google Sheets → AI scoring přes Make.com
Postav scénář v Make: trigger = nový řádek v Sheets → pošli data do Claude/GPT API → zapiš výsledek (skóre, segment, akce) zpět do Sheets do nových sloupců. Michal pak vidí vyhodnocený sheet v reálném čase.
Make: Google Sheets → HTTP → Google Sheets
3
Dashboard načítá data z Google Sheets (živě)
Nastav Google Sheet jako veřejně čitelný (Sdílet → Kdokoli se odkazem → Čtenář) a pak použij Google Sheets JSON API. Dashboard si data stáhne při každém otevření a automaticky vykreslí aktuální stav bez nutnosti uploadovat Excel.
https://sheets.googleapis.com/v4/spreadsheets/SHEET_ID/values/A:Z?key=API_KEY
+
Notifikace při novém vzkazu
V Make přidej podmínku: pokud sloupec "Vzkaz" není prázdný → odešli email na tebe + Michala s obsahem vzkazu a jménem respondenta. Alternativně Slack nebo WhatsApp přes Make modul.
Make: Filter → Email / Slack notify
Stav URL pro Michala (GitHub Pages) Michal může otevřít dashboard kdykoliv přes tvůj GitHub Pages odkaz. Pokud nasadíš verzi s live Google Sheets načítáním, data se aktualizují automaticky — stačí reload stránky. Žádný přístup do Google Sheets nebo Excelu nepotřebuje.
